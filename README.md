# 🚀 Vibe2Vibe - AI Productivity Companion

## The Last-Minute Life Saver
<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/b19fa84d-d14a-456c-825d-18e94661c8b2" />


Vibe2Vibe is an AI-powered productivity companion designed to help students, professionals, and entrepreneurs manage important tasks, prioritize deadlines, and take meaningful action before time runs out.

Unlike traditional reminder apps that only notify users, Vibe2Vibe actively helps users decide **what to do next, when to do it, and how to complete it efficiently** using AI-powered recommendations.

---

## 🌐 Live Demo

Try the deployed application:

🔗 https://ai-studymate-agent.streamlit.app/

---

## 🎯 Problem Statement

People often miss important commitments such as:

* Assignments
* Deadlines
* Meetings
* Bill payments
* Interviews
* Personal goals

Existing productivity tools mainly provide passive reminders, which users often ignore. There is a need for an intelligent assistant that understands priorities and helps users take action.

---

## 💡 Solution

Vibe2Vibe acts as a personal AI productivity assistant that:

✅ Analyzes tasks based on urgency and importance
✅ Provides intelligent task prioritization
✅ Creates actionable productivity plans
✅ Gives personalized recommendations
✅ Helps users avoid last-minute stress

---

# ✨ Key Features

## 🧠 AI Task Prioritization

The AI analyzes user tasks and determines:

* Priority level
* Urgency
* Suggested action plan
* Reason behind the recommendation

---

## 📅 Smart Scheduling Assistance

Helps users organize tasks by suggesting:

* What to complete first
* How to allocate time
* Efficient task ordering

---

## 📊 Productivity Analytics Dashboard

Provides insights into:

* Task completion progress
* Productivity trends
* Performance overview

---

## 🔔 Context-Aware Assistance

Instead of simple reminders, Vibe2Vibe provides meaningful suggestions:

Example:

> "Your deadline is approaching. Start this task now and divide it into smaller steps."

---

## 🤖 AI-Powered Recommendations

Provides personalized productivity suggestions based on user goals and tasks.

# 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/akshayasimhadri/Studymate-AI.git
```

Navigate to the project:

```bash
cd Studymate-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

# 🔐 Environment Variables

Create a `.env` file:

```
GEMINI_API_KEY=your_api_key_here
```

Never upload API keys publicly.
